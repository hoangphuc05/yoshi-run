Project Design

This project is going to feature most prominently a website hosting a side scrolling game, and for our particular group we decided to do a Super Mario type game. Instead of Mario though, the main character will be Yoshi. The design will be as follows:

Essential Elements:
  • Homepage
  • Links (via a menu) to other webpages like an about page and resources used to build the website
  • The game itself on its own page
  • Possibly a comments page where user feedback can be done

The game will be for all intents and purposes will replicate Super Mario, with appropiate ability to run towards an end goal while jumping around, collecting coins, and defeating monsters.
